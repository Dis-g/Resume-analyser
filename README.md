# CVify - AI-Powered Resume Analyzer

**CVify** is a high-performance, full-stack web application designed to help job seekers optimize their resumes for modern ATS (Applicant Tracking Systems). Using cutting-edge AI, CVify provides instant scoring, deep content analysis, and actionable feedback to ensure your CV stands out to recruiters.

## Tech Stack

**Puter.js:** A comprehensive cloud ecosystem providing serverless AI models (GPT-4o-mini), secure Key-Value storage, and a cloud-native file system.

**React Router v7:** The latest evolution of the React framework, offering high-performance Vite-based bundling, server-side rendering capabilities, and simplified routing.

**Tailwind CSS v4:** A utility-first CSS framework used with a custom-built "Executive Dark" theme for a professional, high-trust aesthetic.

**PDF.js:** A powerful library for parsing and rendering PDF documents, enabling CVify to process complex resume layouts into AI-readable formats.

**Zustand:** A small, fast, and scalable bear-necessity state-management solution for handling global application states and Puter.js initialization.

**TypeScript:** Ensures a type-safe development environment, reducing runtime errors and improving codebase maintainability.

## Features

* **Secure Authentication** – Leverage Puter’s built-in cloud identity system for a seamless, passwordless login experience.
* **AI-Driven ATS Scoring** – Instant evaluation of your resume against specific job titles and descriptions using `gpt-4o-mini`.
* **Advanced OCR & PDF Processing** – Converts PDF resumes into optimized images for the AI to "see" layout, formatting, and design.
* **Private History Storage** – Save and manage your previous resume versions securely in Puter’s serverless KV store.
* **Actionable Feedback Loops** – Get specific advice on Tone, Impact, Quantifiable Results, and Keyword Optimization.
